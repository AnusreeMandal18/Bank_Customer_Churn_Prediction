# Bank Customer Churn Prediction

## Project Overview

This project develops a machine learning-based customer churn prediction system for retail banking. The objective is to identify customers who are likely to leave the bank and generate churn risk scores to support proactive retention strategies.

## Objectives

* Predict customer churn with high accuracy
* Generate customer-level churn probability scores
* Identify key factors influencing churn
* Develop an interactive Streamlit dashboard

## Dataset Features

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Ownership
* Active Membership Status
* Estimated Salary

Target Variable:

* Exited (1 = Churned, 0 = Retained)

## Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting
